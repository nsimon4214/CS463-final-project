Random Interval Timer

The Random Interval Timer is a Python-based application designed to bring variety to your workouts or training routines. With customizable minimum and maximum intervals, this timer generates random durations to guide your exercises, rest periods, or other timed activities.

Use Case Example: Set the timer to select intervals between 60 and 120 seconds. Start the session and perform exercises like pushups for 78 seconds, situps for 99 seconds, and squats for 102 seconds, followed by a rest for 68 seconds.

Features:

  • Customizable time intervals.
  • Audio notifications to signal the start and end of each interval.
  • Cross-platform compatibility (Windows, macOS, Linux).
  • Built with Python, featuring a user-friendly GUI.
